Description

With the Vimeo Importer you can import videos from Vimeo to PHPVibe with no effort.

Vimeo Keys

In order for this plugin to work you need to head to developer.vimeo.com and register a new application .
You will only need the:

Client ID (Also known as Consumer Key or API Key)
Client Secret (Also known as Consumer Secret or API Secret)

No upload permission is required.

Installation procedure

Prepare the upload

Unpack the plugin to a folder on your hard drive.

You can safely and fast install this plugin by simply uploading the files contained in /Upload this/ to the server. So, simply copy the entire content of the folder "/Upload this/" to your PHPVibe's root on server. Exclude the folder "/Upload this/" just it's content.

and that's all...

If you have renamed /moderator , also rename it in the plugin before the upload.

Go to Settings -> Plugins and enable this plugin from the list.

You should now see two links:

Settings - > Vimeo Settings (here you add the keys)

Add Media -> Vimeo (this is the actual importer).
